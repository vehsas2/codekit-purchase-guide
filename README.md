## 它到底能干什么

CodeKit 的核心逻辑很简单：**你改代码，它帮你做后续的一切脏活**。

**编译各种语言**。Sass、Less、Stylus、CSS、CoffeeScript、Pug、Slim、Haml、TypeScript、JavaScript、ES6、Markdown、JSON、SVG，以及 PNG、GIF、JPEG 图片——这些全部开箱支持。想压缩 CSS？勾一个框。想把 JS 转译？勾一个框。没有命令行，没有配置文件，就一个干净的界面。

**浏览器即时刷新**。这个功能比你想象的要好用——不是整页刷新，而是 CSS 改动直接注入，页面不重载。意味着你可以直接调那些需要特定状态才能看到的弹窗、抽屉组件的样式，改一行立刻在屏幕上看到效果。更妙的是，Mac、iOS、Android、Windows，只要有现代浏览器都能同步刷新，不需要装任何插件。

**图片优化**。PNG 和 JPEG 文件一键减少超过 70% 体积，SVG、GIF、WebP 也都支持。这个是真实数据，用的是 PNGQuant、Gifsicle、SVGO 这些业界主流工具。手动压图这件事，以后可以彻底忘掉了。

**npm 图形界面**。130 多万个 npm 包，在 CodeKit 里一键安装。能直接看到哪些包过期了，一键更新，不用记命令，不用开终端。

**内置调试**。ESLint 扫描、JSHint、JSLint，找那个不知道藏在哪里的悬挂逗号，在 UI 里就能看到具体是哪一行哪一列出了问题。如果是团队协作，还能设置编码风格规则，强制大家写出统一的代码。
